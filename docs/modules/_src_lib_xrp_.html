reliableBatchPayment

  • reliableBatchPayment(txInputs: TxInput[], txOutputWriteStream: WriteStream, txOutputSchema: z.Schema<TxOutput>, senderWallet: Wallet, xrpClient: XrpClient, usdToXrpRate: number, numRetries: number): Promise<void>
    • Defined in src/lib/xrp.ts:197

    Reliably send a batch of XRP payments from an array of transaction inputs. If any payment fails, exit. As payments succeed, write the output to a CSV. This guarantees that if any payment fails, we will still have a log of succeeded payments (and of course if all payments succeed we will have a log as well).

    Parameters

    • txInputs: TxInput[]

      An array of validated transaction inputs to send payments.

    • txOutputWriteStream: WriteStream

      The write stream.

    • txOutputSchema: z.Schema<TxOutput>

      The output schema.

    • senderWallet: Wallet

      The sender wallet.

    • xrpClient: XrpClient

      The XRP network client.

    • usdToXrpRate: number

      The price of XRP in USD.

    • numRetries: number

      The amount of times to retry a pending payment.

    Returns Promise<void>
